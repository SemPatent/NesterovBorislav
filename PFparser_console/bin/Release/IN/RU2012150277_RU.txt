
1. Устройство, содержащее:
- приемный блок, выполненный с возможностью принимать первое пространственное положение, соответствующее первой части человеческого         тела, и второе пространственное положение, соответствующее второй части человеческого тела;      
- идентифицирующий блок, выполненный с возможностью идентифицировать группу объектов на основе по меньшей мере первого пространственного         положения; и      
- выбирающий блок, выполненный с возможностью выбирать объект из идентифицированной группы на основе второго пространственного         положения.      
2. Устройство по п.1, в котором первая часть человеческого тела дистальна по отношению к левому плечу, а вторая часть человеческого         тела дистальна по отношению к правому плечу.      
3. Устройство по п.1, в котором:
- первое пространственное положение, соответствующее первой опорной точке, расположенной рядом с первой частью человеческого         тела; и      
- второе пространственное положение, соответствующее второй опорной точке, расположенной рядом со второй частью человеческого         тела.      
4. Устройство по п.3, содержащее:
- блок, выполненный с возможностью находить в базе данных информацию о позе, соответствующую первой и второй частям человеческого         тела, причем информация о позе содержит множество пространственных положений, соответствующих первой опорной точке и второй         опорной точке.      
5. Устройство по п.4, дополнительно содержащее:
- определяющий блок, выполненный с возможностью определять, соответствует ли первое пространственное положение первому жесту,         на основе по меньшей мере найденной информации о позе.      
6. Устройство по п.5, в котором определяющий блок дополнительно выполнен с возможностью:
- сравнивать первое пространственное положение с информацией о позе, соответствующей первой опорной точке; и
- определять, что первое пространственное положение соответствует первому жесту, когда первое пространственное положение соответствует         по меньшей мере одному из пространственных положений информации о позе, соответствующей первой опорной точке.      
7. Устройство по п.5, в котором идентифицирующий блок дополнительно выполнен с возможностью:
- присваивать первую команду первому пространственному положению, когда первое пространственное положение соответствует первому         жесту.      
8. Устройство по п.7, в котором идентифицирующий блок дополнительно выполнен с возможностью:
- идентифицировать группу объектов в соответствии с первой командой.
9. Устройство по п.4, в котором идентифицирующий блок дополнительно выполнен с возможностью:
- определять характеристику первого жеста на основе сравнения первого пространственного положения с по меньшей мере одним         пространственным положением информации о позе, которая соответствует первой опорной точке.      
10. Устройство по п.9, в котором характеристика содержит по меньшей мере один из следующих параметров: скорость, смещение         или угол поворота.      
11. Устройство по п.9, в котором идентифицирующий блок дополнительно выполнен с возможностью:
- идентифицировать группу объектов на основе по меньшей мере первого пространственного положения и характеристики первого         жеста.      
12. Устройство по п.5, в котором идентифицирующий блок дополнительно выполнен с возможностью:
- присваивать обобщенную команду первому пространственному положению, когда первое пространственное положение не соответствует         первому жесту.      
13. Устройство по п.5, в котором идентифицирующий блок дополнительно выполнен с возможностью:
- определять, соответствует ли второе пространственное положение второму жесту, на основе по меньшей мере найденной информации         о позе.      
14. Устройство по п.13, в котором определяющий блок дополнительно выполнен с возможностью:
- сравнивать второе пространственное положение с информацией о позе, соответствующей второй опорной точке; и
- определять, что второе пространственное положение соответствует второму жесту, когда второе пространственное положение соответствует         по меньшей мере одному из пространственных положений информации о позе, соответствующей второй опорной точке.      
15. Устройство по п.14, в котором выбирающий блок дополнительно выполнен с возможностью:
- присваивать вторую команду второму пространственному положению, когда второе пространственное положение соответствует второму         жесту.      
16. Устройство по п.15, в котором выбирающий блок дополнительно выполнен с возможностью:
- выбирать объект идентифицированной группы на основе по меньшей мере второй команды.
17. Устройство по п.1, дополнительно содержащее:
- блок создания изображений, выполненный с возможностью фиксировать изображение, содержащее по меньшей мере первую и вторую         часть человеческого тела.      
18. Устройство по п.17, в котором приемный блок дополнительно выполнен с возможностью:
- обрабатывать зафиксированное изображение, чтобы идентифицировать первое пространственное положение и второе пространственное         положение.      
19. Устройство по п.1, дополнительно содержащее:
- блок, выполненный с возможностью выполнять функцию, соответствующую отобранному объекту.
20. Способ управления интерфейсом с помощью жестов, реализуемый с помощью компьютера и содержащий этапы, на которых:
- принимают первое пространственное положение, соответствующее первой части человеческого тела, и второе пространственное         положение, соответствующее второй части человеческого тела;      
- идентифицируют группу объектов на основе по меньшей мере первого пространственного положения; и
- выбирают с помощью процессора объект из идентифицированной группы на основе по меньшей мере второго пространственного положения.

- принимают первое пространственное положение, соответствующее первой части человеческого тела, и второе пространственное         положение, соответствующее второй части человеческого тела;      
- идентифицируют группу объектов на основе по меньшей мере первого пространственного положения; и
- выбирают с помощью процессора объект из идентифицированной группы на основе по меньшей мере второго пространственного положения.
