1.            Вентиляционное устройство для окон с функцией очистки воздуха, содержащее:      
теплообменный блок, установленный на оконной и дверной рамах с возможностью осуществления теплообмена между наружным воздухом         и воздухом закрытых помещений; и часть для очистки воздуха, которая выполнена в теплообменном блоке для очищения воздуха закрытых         помещений.      
2. Вентиляционное устройство по п.1, в котором теплообменный блок содержит:
приемную коробку;
вентилятор для воздуха закрытых помещений, расположенный на одной стороне коробки с возможностью ввода воздуха закрытых помещений,         выпускаемого в наружное пространство, в коробку, а также его выпуска из коробки к оконной и дверной раме;      
вентилятор для наружного воздуха, расположенный на другой стороне коробки с возможностью ввода наружного воздуха из оконной         и дверной рам в коробку, а также его выпуска из коробки в наружное пространство коробки; и      
теплообменное устройство, расположенное на центральном участке коробки для осуществления теплообмена между воздухом закрытых         помещений и наружным воздухом.      
3. Вентиляционное устройство по п.2, в котором впускной канал для наружного воздуха, через который вводится наружный воздух,         выполнен на одной стороне коробки.      
4. Вентиляционное устройство по п.3, в котором выпускной канал для воздуха закрытых помещений, через который воздух закрытых         помещений выпускается, выполнен на другой стороне коробки.      
5. Вентиляционное устройство по п.4, в котором выпускной канал для выпуска наружного воздуха выполнен на одном конце коробки         с тем, чтобы быть перпендикулярным выпускному каналу для воздуха закрытых помещений.      
6. Вентиляционное устройство по п.5, в котором впускное отверстие для воздуха закрытых помещений, через которое воздух закрытых         помещений вводится в коробку, выполнено на внутренней поверхности коробки.      
7. Вентиляционное устройство по п.2, в котором теплообменное устройство содержит:
первое теплообменное устройство, которое первоначально осуществляет теплообмен между воздухом закрытых помещений и наружным         воздухом; и      
второе теплообменное устройство, которое дополнительно осуществляет теплообмен между воздухом закрытых помещений и наружным         воздухом, проходящим через первое теплообменное устройство.      
8. Вентиляционное устройство по п.7, в котором первое и второе теплообменные устройства - все являются теплообменными устройствами         явной теплоты, или энтальпийными теплообменными устройствами, или сочетанием теплообменного устройства и энтальпийного теплообменного         устройства.      
9. Вентиляционное устройство по п.2, дополнительно содержащее фильтр для наружного воздуха, расположенный в коробке для очищения         наружного воздуха, перед тем как наружный воздух проходит через теплообменное устройство.      
10. Вентиляционное устройство по п.2, дополнительно содержащее фильтр для воздуха закрытых помещений, расположенный в коробке         для очищения воздуха закрытых помещений, перед тем как воздух закрытых помещений проходит через теплообменное устройство.      
11. Вентиляционное устройство по любому из пп.5-7 или 10, в котором часть для очистки воздуха содержит:
второй фильтр для наружного воздуха, выполненный в выпускном канале для наружного воздуха;
третий фильтр для наружного воздуха, расположенный на месте, противоположном фильтру для наружного воздуха; и
второй фильтр для воздуха закрытых помещений, расположенный на одной стороне первого теплообменного устройства с тем, чтобы         быть напротив фильтра для воздуха закрытых помещений.      
12. Вентиляционное устройство по п. 3 или 5, в котором
заслонка, поворачивающаяся в одном или в другом направлении с помощью двигателя переднего и реверсивного хода, с тем чтобы         открывать и закрывать впускной канал для наружного воздуха и выпускной канал для наружного воздуха, выполнена на внутренней         стороне впускного канала для наружного воздуха и выпускного канала для наружного воздуха.      
13. Вентиляционное устройство по п.12, дополнительно содержащее блок управления, контролирующий прямое и реверсивное вращение         двигателя.      
14. Вентиляционное устройство по п.13, в котором блок управления соединен электрическим образом с вентилятором для наружного         воздуха теплообменного блока.
