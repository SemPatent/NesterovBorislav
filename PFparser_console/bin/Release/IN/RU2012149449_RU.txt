1. Узел оценки величины генерирования тепла для батареи для электрического инструмента, который обеспечен в устройстве для         электрического инструмента и оценивает величину генерирования тепла батареи, которая представляет собой источник питания электрического         инструмента, при этом узел содержит:      
вычислительное устройство, которое периодически считывает, либо во время разрядки от батареи, либо во время зарядки на батарею,         детектированный ток от устройства детектирования тока, которое детектирует ток, соответствующий этому моменту времени, из         тока разряда, протекающего от батареи, и тока заряда, протекающего на батарею, и прибавляет/вычитает эквивалентное значение         величины генерирования тепла в соответствии со значением считанного детектированного тока,      
в котором вычислительное устройство выдает прибавленное/вычтенное эквивалентное значение величины генерирования тепла в виде         оцененного значения, представляющего собой величину генерирования тепла батареи.      
2. Узел оценки величины генерирования тепла для батареи для электрического инструмента по п.1,
в котором вычислительное устройство содержит устройство установки прибавляемого/вычитаемого значения, которое устанавливает         прибавляемое/вычитаемое значение таким образом, что когда значение детектированного тока меньше, чем предопределенное пороговое         значение, отрицательное прибавляемое/вычитаемое значение, в том числе ноль, устанавливается таким образом, что, чем меньше         значение детектированного тока, тем больше абсолютное значение; и, когда значение детектированного тока равняется или больше,         чем предопределенное пороговое значение, положительное прибавляемое/вычитаемое значение устанавливается таким образом, что,         чем больше значение детектированного тока, тем больше прибавляемое/вычитаемое значение, и      
в котором эквивалентное значение величины генерирования тепла обновляется посредством прибавления отрицательного прибавляемого/вычитаемого         значения или положительного прибавляемого/вычитаемого значения, установленного устройством установки прибавляемого/вычитаемого         значения, к эквивалентному значению величины генерирования тепла.      
3. Узел оценки величины генерирования тепла для батареи для электрического инструмента по п.2,
в котором устройство установки прибавляемого/вычитаемого значения устанавливает прибавляемое/вычитаемое значение, основываясь         на значении детектированного тока и детектированной температуры от устройства детектирования температуры, которое детектирует         температуру батареи, таким образом, что, чем больше значение детектированного тока и чем выше детектированная температура,         тем больше прибавляемое/вычитаемое значение.      
4. Устройство для электрического инструмента, содержащее:
узел оценки величины генерирования тепла для батареи для электрического инструмента по любому из п.п.1-3;
устройство определения, которое определяет, превысило ли оцененное значение, представляющее собой величину генерирования тепла         батареи, полученное посредством узла оценки величины генерирования тепла, установленное значение для определения перегрева,         установленное заранее; и      
защитное устройство, которое защищает батарею посредством выполнения, если устройством определения определено, что оцененное         значение превысило установленное значение, любой из обработки приостановки, приостанавливающей работу, соответствующую этому         моменту времени, из разрядки от батареи и зарядки на батарею, и обработки ограничения, ограничивающей работу, соответствующую         этому моменту времени, из разрядки от батареи и зарядки на батарею.      
5. Устройство для электрического инструмента по п.4,
в котором, если устройством определения определено, что оцененное значение превысило установленное значение, защитное устройство         ограничивает любое из разрядки от батареи и зарядки на батарею посредством коррекции управляющего порогового значения, использующегося         либо для управления разрядкой батареи, либо для управления зарядкой батареи.      
6. Устройство для электрического инструмента по п.5,
в котором, если устройством определения определено, что оцененное значение превысило установленное значение либо во время         разрядки от батареи, либо во время зарядки на батарею, защитное устройство корректирует управляющее пороговое значение, использующееся         либо во время следующей зарядки на батарею, либо во время следующей разрядки от батареи.      
7. Устройство для электрического инструмента по любому из п.п.4-6,
в котором устройство представляет собой любое из батарейного блока, содержащего батарею в нем, основного корпуса электрического         инструмента, к которому батарейный блок съемно прикрепляется, и зарядного устройства батареи, к которому батарейный блок съемно         прикрепляется.
