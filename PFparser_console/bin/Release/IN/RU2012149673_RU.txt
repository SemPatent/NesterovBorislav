
1. Способ отображения текстовой информации, отличающийся тем, что:
- отображают список из по меньшей мере одного элемента, представляющего текстовую информацию и частичное содержимое текстовой         информации; и      
- изменяют список в ответ на заданный клавиатурный ввод, чтобы продемонстрировать все содержимое текстовой информации.
2. Способ по п.1, в котором список содержит первую пиктограмму и этап изменения содержит этап, на котором, когда производится         прикосновение к первой пиктограмме, расширяют по меньшей мере один элемент в заданном направлении, чтобы представить полное         содержимое текстовой информации.      
3. Способ по п.2, в котором этап изменения содержит этап, на котором заменяют первую пиктограмму второй пиктограммой, при         этом способ дополнительно содержит этапы, на которых:      
- выполняют отслеживание, чтобы обнаружить ввод в отношении второй пиктограммы; и
- сжимают по меньшей мере один элемент, когда обнаружен ввод в отношении второй пиктограммы, в противоположном направлении,         чтобы представить частичное содержимое текстовой информации.      
4. Способ по п.1, в котором по меньшей мере один элемент содержит символ пропуска.
5. Способ по п.2, в котором этап отображения содержит этапы, на которых:
- извлекают частичное содержимое из текстовой информации;
- упорядочивают по меньшей мере один элемент, имеющий третью пиктограмму, в заданном направлении; и
- представляют список из по меньшей мере одного элемента, имеющего частичное содержимое, вместе с первой пиктограммой.

Способ по п.5, в котором этап изменения содержит этап, на котором, когда обнаружен ввод в отношении третьей пиктограммы,         расширяют элемент, имеющий третью пиктограмму, в отношении которой обнаружен ввод, в заданном направлении, чтобы представить         полное содержимое текстовой информации, и, когда обнаружен ввод в отношении первой пиктограммы, по меньшей мере один элемент         в направлении, чтобы представить полное содержимое текстовой информации.      
7. Способ по п.6, в котором этап изменения содержит этап, на котором заменяют третью пиктограмму четвертой пиктограммой и         заменяют первую пиктограмму второй пиктограммой.      
8. Способ отображения текстовой информации, содержащий этапы, на которых:
- отображают список множества элементов текстовой информации и части текстовой информации для каждого из множества элементов;
- выбирают по меньшей мере один из элементов в соответствии со вводом пользователя; и
- удаляют упомянутый по меньшей мере один выбранный элемент при вводе заданной клавишей.
9. Способ по п.8, в котором этап отображения содержит этап, на котором представляют первую пиктограмму в списке и представляют         вторую пиктограмму в каждом из элементов, когда обнаружен ввод в отношении первой пиктограммы.      
10. Способ по п.9, в котором этап выбора содержит этап, на котором изменяют форму второй пиктограммы по меньшей мере одного         выбранного элемента, и этап отображения содержит этап, на котором конфигурируют каждый элемент с полным содержимым текстовой         информации, упорядочивают элементы, имеющие третьи пиктограммы, в заданном направлении, и представляют список, имеющий первую         пиктограмму, в области.      
11. Способ по п.10, дополнительно содержащий этап, на котором заменяют третьи пиктограммы элементов на вторые пиктограммы,         когда обнаружен ввод в отношении первой пиктограммы.      
12. Способ по п.9, в котором этап представления второй пиктограммы содержит этап, на котором сжимают элементы, чтобы показать         частичное содержимое соответствующей текстовой информации, и упорядочивают элементы в направлении, противоположном направлению         расширения.      
13. Способ по п.10, дополнительно содержащий этап, на котором удаляют элемент, имеющий выбранную третью пиктограмму, из списка,         когда выбрана по меньшей мере одна из третьих пиктограмм.
