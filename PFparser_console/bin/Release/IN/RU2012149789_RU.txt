1. Компьютерный программный продукт для измерения использования ресурсов в вычислительной среде, причем указанный компьютерный         программный продукт включает:      
энергонезависимую машиночитаемую запоминающую среду, считываемую устройством обработки и хранящую команды для выполнения устройством         обработки для выполнения способа, включающего:      
выборочную регистрацию функции PCI для измерения, причем выборочная регистрация включает выполнение команды изменения управления         функцией шины PCI (Modify PCI Function Controls, MPFC), для регистрации функции PCI для измерения, при этом функция PCI идентифицируется         в команде MPFC дескриптором функции, причем дескриптор функции содержит номер функции, идентифицирующий функцию PCI, где номер         функции сконфигурирован адресовать функцию PCI и служить индексом в таблице функций, которая содержит один или более признаков         функции PCI, и причем команда MPFC определяет местоположение в памяти для информации слежения для функции PCI;      
определение того, что (а) к функции PCI осуществляется доступ посредством команды, или того, что (b) функция PCI запрашивает         доступ к системной памяти, связанной с функцией PCI;      
отслеживание информации, характерной для функции PCI, причем информация слежения включает подсчет определенных осуществляющих         доступ команд или подсчет определенных запрошенных операций доступа к памяти, и причем информация отслеживается на основе         отдельных функций; и сохранение информации слежения в местоположении в памяти.      
2. Компьютерный программный продукт по п.1, отличающийся тем, что при указанном определении определяют, что к функции PCI         осуществляется доступ посредством команды, при этом отслеживание также содержит:      
определение типа команды, причем имеется множество типов команд; и
обновление счетчика, предназначенного исключительно для этого типа команды и для функции PCI, причем другой счетчик предоставляется         для другого типа команды для функции PCI.

Компьютерный программный продукт по п.2, отличающийся тем, что имеется множество типов команд, включая команду загрузки         (PCI Load), команду сохранения (PCI Store), команду сохранения блока (PCI Store Block), команду изменения (PCI Modify) для         выполнения операции изменения для функции PCI и команду обновления трансляции (PCI Refresh Translation), для обновления информации,         используемой функцией PCI, каждая из которых специально используется для доступа к функции PCI.

определение типа операции доступа к памяти и
обновление счетчика, предназначенного исключительно для функции PCI и этого типа операции доступа к памяти.
