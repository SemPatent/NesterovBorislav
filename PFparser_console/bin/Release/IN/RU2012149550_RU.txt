1. Устройство для тиснения для нанесения тиснения на обертку курительного изделия, выполненное с возможностью нанесения рельефа         тиснения на обертку с обеспечением по меньшей мере частичного сведения к минимуму снижения по меньшей мере в одном направлении,         прочности на разрыв обертки, после нанесения тиснения.      
2. Устройство для тиснения по п.1, обеспечивающее нанесение рельефа тиснения, так что увеличивается коэффициент диффузии и         (или) проницаемость обертки, и при этом рельеф тиснения выбирается таким, чтобы для данного увеличения коэффициента диффузии         и (или) проницаемости, рельеф тиснения сводил к минимуму по меньшей мере в одном направлении напряжение тиснения и (или) растяжение,         воздействующие на обертку, в процессе нанесения рельефа тиснения.      
3. Устройство для тиснения по п.2, обеспечивающее нанесение рельефа тиснения, так что после его нанесения тисненая обертка         обладает различной прочностью в разных направлениях.      
4. Устройство для тиснения по любому из предшествующих пунктов, содержащее:
гравировальный валик для нанесения тиснения на обертку и
ограничитель перемещения, выполненный с возможностью поддержания заданного расстояния между гравировальным валиком и оберткой         для ограничения глубины тиснения.      
5. Устройство для тиснения по п.4, содержащее по меньшей мере один дополнительный гравировальный валик для нанесения тиснения         на обертку, проходящую в продольном направлении между по меньшей мере двумя валиками, при этом ограничитель перемещения выполнен         с возможностью поддерживать минимальное расстояние между круговыми поверхностями по меньшей мере двух гравировальных валиков.      
6. Тисненая обертка для курительного изделия с нанесенным рельефом тиснения, выбранным таким образом, что для данного увеличения         коэффициента диффузии и(или) проницаемости в результате тиснения нанесенный рельеф по меньшей мере частично сводит к минимуму         снижение по меньшей мере в одном направлении прочности на разрыв обертки после нанесения тиснения.      
7. Тисненая обертка по п.6, в которой рельеф тиснения выбран так, чтобы свести к минимуму линии ослабления прочности, проходящие         через область тиснения обертки, по меньшей мере, в одном направлении.      
8. Тисненая обертка по п.6 или 7, в которой рельеф тиснения включает область тиснения и нетисненую область и переднюю кромку         между областью тиснения и нетисненой областью, при этом передняя кромка располагается неперпендикулярно продольному направлению         обертки.      
9. Тисненая обертка по п.6 или 7, в которой рельеф тиснения включает спиральный или винтовой узор, когда обертка обернута         вокруг курительного изделия.      
10. Тисненая обертка по п.6 или 7, в которой рельеф тиснения обертки содержит углубления, не имеющие кромок, по существу перпендикулярных         относительно продольного направления обертки.      
11. Тисненая обертка по п.6 или 7, в которой рельеф тиснения обертки содержит углубления ромбовидной формы, не имеющие кромок,         по существу перпендикулярных относительно продольного направления обертки.      
12. Тисненая обертка по п.6 или 7, в которой рельеф тиснения сформирован из решетки углублений, в которой поперечно смежные         углубления смещены друг относительно друга в продольном направлении.      
13. Гравировальный валик для нанесения рельефа тиснения на обертку курительного изделия, имеющий группу гравировальных выступов,         расположенных таким образом, что при данном увеличении коэффициента диффузии и (или) проницаемости обертки, тисненной с использованием         этого валика, гравировальные выступы по меньшей мере частично сводят к минимуму снижение по меньшей мере в одном направлении         прочности на разрыв обертки в результате нанесения этого тиснения.      
14. Гравировальный валик по п.13, в котором гравировальные выступы расположены таким образом, что при приложении гравировального         валика к обертке обеспечивается обертка по п.6 или 7.      
15. Способ тиснения обертки для курительного изделия, при выполнении которого наносят рельеф тиснения на обертку так, что         по меньшей мере частично сводят к минимуму снижение по меньшей мере в одном направлении прочности на разрыв обертки в результате         нанесения этого тиснения.      
16. Способ по п.15, в котором при нанесении тиснения накладывают гравировальный валик на обертку и поддерживают гравировальный         валик на заданном расстоянии от обертки для ограничения глубины тиснения.      
17. Способ по п.15 или 16, в котором при нанесении тиснения протягивают ленту материала обертки в продольном направлении через         по меньшей мере два гравировальных валика, а для поддержания гравировального валика на заданном расстоянии используют коаксиальный         кулачковый ограничитель перемещения и механический упор для предотвращения радиального смещения по меньшей мере одного из         валиков в по меньшей мере одном направлении.      
18. Курительное изделие, содержащее штранг курительного материала и обертку по п.6 или 7.
19. Устройство для тиснения для нанесения тиснения на обертку курительного изделия, выполненное с возможностью нанесения на         обертку рельефа тиснения, включающего область тиснения, нетисненую область и переднюю кромку между областью тиснения и нетисненой         областью, причем передняя кромка располагается неперпендикулярно продольному направлению обертки так, что обеспечивается по         меньшей мере частичное сведение к минимуму снижения по меньшей мере в одном направлении прочности на разрыв обертки после         нанесения тиснения.      
20. Тисненая обертка для курительного изделия, имеющая рельеф тиснения, включающий область тиснения, нетисненую область и         переднюю кромку между областью тиснения и нетисненой областью, причем передняя кромка располагается неперпендикулярно продольному         направлению обертки так, что для данного увеличения коэффициента диффузии и(или) проницаемости в результате тиснения нанесенный         рельеф по меньшей мере частично сводит к минимуму снижение по меньшей мере в одном направлении, прочности на разрыв обертки         после нанесения тиснения.
