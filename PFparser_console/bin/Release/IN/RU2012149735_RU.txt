1. Устройство обработки информации, содержащее:
блок связи, предназначенный для выполнения прямого общения в ближнем поле с первым устройством связи;
блок получения информации, предназначенный для получения информации содержания, хранящейся в первом устройстве связи посредством         связи с первым устройством связи;      
блок обработки отображения, предназначенный для выполнения обработки отображения для информации, полученной блоком получения         информации; и      
блок отображения, предназначенный для отображения информации содержания.
2. Устройство обработки информации по п.1,
в котором информация содержания, хранящаяся в первом устройстве связи, является содержанием с движущимся изображением, воспроизводимым         первым устройством связи, и      
в котором блок отображения отображает содержание с движущимся содержанием, которое воспроизводит первое устройство связи.
3. Устройство обработки информации по п.1, дополнительно содержащее:
блок передачи информации, предназначенный для передачи информации содержания, хранящейся в первом устройстве связи, на второе         устройство связи,      
и в котором блок связи выполняет прямое общение в ближнем поле со вторым устройством связи в дополнение к первому устройству         связи.      
4. Устройство обработки информации по п.3, дополнительно содержащее:
блок обнаружения операции, предназначенный для обнаружения операции с блоком отображения,
в котором блок передачи информации передает информацию содержания, когда обнаружена операция с информацией содержания, отображаемой         на блоке отображения.      
5. Устройство обработки информации по п.3,
в котором блок передачи информации передает информацию содержания, хранящуюся в первом устройстве связи, с преобразованием         формата.      
6. Устройство обработки информации по п.3,
в котором блок передачи информации передает информацию содержания, которую не хранит первое устройство связи среди содержания,         которое хранит непосредственно само устройство.      
7. Устройство обработки информации по п.1, дополнительно содержащее:
блок обнаружения расположения, предназначенный для обнаружения расположения первого и второго устройства связи,
в котором блок передачи информации изменяет информацию содержания, которая должна быть передана на второе устройство связи,         в соответствии с расстоянием между первым устройством связи и вторым устройством связи.      
8. Устройство обработки информации по п.3,
в котором блок передачи информации передает информацию содержания так, что соотношение для синхронизации информации содержания         первого устройства связи и второго устройства связи увеличивают, когда расстояние между первым устройством связи и вторым         устройством связи становится короче.      
9. Система обработки информации, содержащая:
первое устройство, предназначенное для выполнения связи со вторым устройством, используя прямое общение в ближнем поле; и
второе устройство, которое включает в себя блок связи, выполняющий прямое общение в ближнем поле с первым устройством, блок         получения информации, предназначенный для получения информации содержания, хранящейся в первом устройстве посредством связи         с первым устройством, блок обработки отображения, предназначенный для выполнения процесса отображения информации содержания,         полученной блоком получения информации, и блок отображения, предназначенный для отображения информации содержания.      
10. Считываемый компьютером носитель информации, содержащий записанную на нем программу, позволяющую компьютеру функционировать         как:      
средство для выполнения прямого общения в ближнем поле с другим устройством связи;
средство для получения информации содержания, сохраненной в первом устройстве связи, посредством связи с другим устройством         связи; и      
средство для выполнения процесса отображения информации содержания.
