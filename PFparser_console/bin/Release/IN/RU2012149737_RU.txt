1. Устройство обработки информации, содержащее:
модуль связи для осуществления взаимно однозначной связи малого радиуса с устройством противоположной стороны связи;
модуль получения состояния связи для получения состояния связи с устройством противоположной стороны связи;
модуль обработки отображения для выполнения обработки отображения для выполнения указания в направлении предпочтительного         состояния связи на основе полученного состояния связи; и      
модуль отображения для выполнения отображения для указания.
2. Устройство обработки информации по п.1,
в котором модуль связи содержит множество антенн, при этом
модуль получения состояния связи выполнен с возможностью получения состояния связи с устройством противоположной стороны связи,         основанного на интенсивности принимаемого сигнала, получаемого каждой из множества антенн.      
3. Устройство обработки информации по п.1,
в котором модуль обработки отображения выполнен с возможностью обработки для отображения на модуле отображения, указания направления         предпочтительного состояния связи.      
4. Устройство обработки информации по п.1, дополнительно содержащее:
модуль вывода, выполненный с возможностью вывода звука или вибрации, способствующие указанию в направлении, предпочтительного         состояния связи.      
5. Система обработки информации, содержащая:
первое устройство, выполненное с возможностью осуществления взаимно однозначной связи, с использованием связи малого радиуса         со вторым устройством, и      второе устройство, содержащее модуль связи для осуществления связи малого радиуса от одного к другому с первым устройством,         модуль получения состояния связи для получения состояния связи с первым устройством, модуль обработки отображения для выполнения         обработки отображения для выполнения указания в направлении предпочтительного состояния связи, на основе полученного состояния         связи, и модуль отображения для выполнения отображения указания.
