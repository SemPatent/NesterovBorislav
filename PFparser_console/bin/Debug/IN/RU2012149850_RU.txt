1. Система двойного цилиндра, содержащая:
по меньшей мере одно устройство управления, выполненное с возможностью передавать управляющие сигналы, при этом устройство         управления далее содержит:      
управляющую полость и
первый поршень, расположенный внутри управляющей полости, при этом первый поршень делит управляющую полость на первую часть         управляющей полости и вторую часть управляющей полости и препятствует сообщению между первой частью управляющей полости и         второй частью управляющей полости;      
по меньшей мере один исполнительный механизм, находящийся в сообщении с устройством управления и выполненный с возможностью         реагировать на управляющие сигналы, передаваемые устройством управления, при этом исполнительный механизм, содержит:      
исполнительную полость и
второй поршень, расположенный внутри исполнительной полости, при этом второй поршень делит исполнительную полость на первую         часть исполнительной полости и вторую часть исполнительной полости и препятствует сообщению между первой частью исполнительной         полости и второй частью исполнительной полости; и      
по меньшей мере одну линию управления, обеспечивающую сообщение между первой частью управляющей полости и первой частью исполнительной         полости; и      
по меньшей мере одну линию управления, обеспечивающую сообщение между второй частью управляющей полости и второй частью исполнительной         полости.      
2. Система по п. 1, далее содержащая манипулятор, в которой манипулятор выполнен с возможностью менять положение первого поршня         внутри управляющей полости.      
3. Система по п. 2, в которой манипулятор соединен с устройством телеманипуляции.
4. Система по п.1, в которой управляющие сигналы генерируются устройством телеманипуляции.
5. Хирургическое устройство, содержащее:
по меньшей мере одно устройство управления, расположенное на проксимальном конце устройства, при этом устройство управления         выполнено с возможностью передавать управляющие сигналы;      
по меньшей мере один манипулятор, при этом манипулятор выполнен с возможностью управления рукой человека и с возможностью         приводить в действие устройство управления;      
по меньшей мере один исполнительный механизм, расположенный на дистальном конце устройства, при этом исполнительный механизм         сообщается с устройством управления и выполнен с возможностью реагировать на управляющие сигналы, передаваемые устройством         управления; и      
по меньшей мере одну линию управления, обеспечивающую сообщение между устройством управления и исполнительным механизмом.
6. Хирургическое устройство по п. 5, в котором манипулятор удален от устройства управления.
7. Хирургическое устройство по п. 6, в котором манипулятор соединен с устройством управления устройством телеманипуляции.
8. Хирургическое устройство по п. 5, в котором управляющие сигналы генерируются устройством телеманипуляции.
9. Хирургическое устройство, содержащее:
управляющую часть, расположенную на проксимальном конце устройства, содержащую:
множество устройств управления, каждое из которых выполнено с возможностью передавать управляющие сигналы; и
множество манипуляторов, каждый из которых выполнен с возможностью приводить в действие соответствующее устройство из множества         устройств управления;      
исполнительную часть, расположенную на дистальном конце устройства, содержащую:
множество исполнительных механизмов, каждый из которых сообщается с соответствующим устройством из множества устройств управления         и выполнено с возможностью реагировать на управляющие сигналы, передаваемые соответствующим устройством из множества устройств         управления; и      
промежуточную часть, содержащую множество линий управления, каждая из которых обеспечивает сообщение с одним устройством из         множества устройств управления и соответствующим одним механизмом из множества исполнительных механизмов.      
10. Хирургическое устройство по п. 9, в котором по меньшей мере одно из устройств управления удалено от множества манипуляторов.
11. Хирургическое устройство по п. 9, в котором по меньшей мере одно из устройство управления соединено с одним из множества         манипуляторов с помощью устройства телеманипуляции.      
12. Хирургическое устройство по п. 9, в котором управляющие сигналы генерируются устройством телеманипуляции.
13. Хирургическое устройство по п. 9, в котором сообщение осуществляется через непосредственное механическое соединение.
14. Хирургическое устройство по п. 9, в котором сообщение осуществляется через опосредованное механическое соединение.
15. Хирургическое устройство по п. 9, в котором сообщение осуществляется через устройство телеманипуляции.
